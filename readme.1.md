# 🚀 FairAI Bias Analyzer

> 📌 This project demonstrates an AI-powered fairness analysis system with bias detection, explanation, and mitigation simulation.

---

## 🧠 Overview
FairAI Bias Analyzer detects bias in datasets and helps users understand and improve fairness using AI (Google Gemini).

---

## 🎯 Problem Statement
Machine learning systems can produce biased outcomes. Existing tools provide only numerical metrics, making it difficult for non-experts to understand and fix bias.

---

## 💡 Solution
- Detect bias using fairness metrics
- Explain bias using AI
- Simulate mitigation to improve fairness

---

## ⚙️ Features

### Core
- Upload CSV dataset
- Select target & sensitive column
- Selection rate calculation
- Disparate impact
- Bias detection

### Visualization
- Bar chart (group comparison)

### AI Features (Gemini)
- Bias explanation
- Affected group identification
- Recommendations

### 🚀 Mitigation Simulation
- Slider-based fairness adjustment
- Recalculated impact
- Before vs after comparison

---

## 🔄 Workflow

Upload → Analyze → Visualize → AI Insight → Mitigation Simulation

---

## 🧩 Tech Stack
- Frontend: HTML, CSS, JS
